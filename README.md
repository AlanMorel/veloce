# vue3-ts-vite-ssr-starter

- Lightning-fast cold server start
- Instant hot module replacement (HMR) and dev SSR
- True on-demand compilation
- Use `vue3 vite2.7 typescript eslint SSR pinia vue-router scss`

# Getting Started

- dev

```bash
npm i
npm run dev
```

- preview

```bash
npm i
npm run preview
```

- build

```bash
npm i
npm run build
```
